TLED

Matter-over-Thread LED Controller

1. Install
2. Configure

Install Firmware

Connect your ESP32-C6 via USB and click Install. After installation, go to Configure tab to set up your LED strip.

Your browser doesn't support Web Serial. Use Chrome or Edge. Serial access not allowed. Make sure you're using HTTPS.

After Installation

Go to the Configure tab to set your LED count, GPIO pin, and other settings. Then scan the QR code from the serial output with your smart home app.
