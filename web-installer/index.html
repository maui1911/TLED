TLED

Matter-over-Thread LED Controller

What is TLED?

TLED turns your ESP32-C6 into a smart LED strip controller that works with Apple Home, Google Home, and other Matter-compatible platforms over Thread mesh networking.

How It Works

Install Firmware

Connect your ESP32-C6 via USB and click Install below

Commission Device

Scan the QR code with your smart home app

Control Your LEDs

Adjust color, brightness, and effects from any Matter app

Configuration

Note: Configuration is set at compile time. This installer provides pre-built firmware with common configurations. For custom setups, build from source using idf.py menuconfig.

After Installation

Open a serial monitor to see the QR code URL for commissioning:

  1. Keep the device connected via USB
  2. Open the browser's serial console or use a terminal app
  3. Look for the QR code URL in the output
  4. Scan the QR code with your smart home app
